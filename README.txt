== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip `bonjoro.zip` into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert your App ID and Identity Verification Secret Key from
http://www.bonjoro.com/settings/web-delivery
